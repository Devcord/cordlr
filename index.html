About

Cordlr is a modular approach for Discord Bots so Server owners can put together their own bots without coding any single line. The only requirement is a hosting service which is capable to host NodeJS apps with version 6.0 and higher.

Installation

Requirements

  • Node.js 6 >
  • Discord

Installation

In this demonstration we're using nave for managing the node environment.

  • nave use 6 or nave use auto - installs latest node version for major 6 or the naverc version
  • npm install cordlr-cli -g - installs the cordlr binary globally or npm install -g devcord/cordlr-cli for the latest stable commit

If you want to learn more about setting up a bot, consider reading our Get Started guide.

We also provide a server boilerplate so you can get started with your bot even faster.

Plugins

We already have a small list of plugins developed and ready for your server! Check them out here. You can install them for example via
npm i cordlr-roles --save

  • cordlr-roles - Role Manager for Cordlr
  • cordlr-ddg - DuckDuckGo Search for Cordlr
  • cordlr-kontrolla - Bot Control Plugin for Cordlr
  • cordlr-giphy - Random GIPHY links for Cordlr
  • cordlr-cats - Random Cat images for Cordlr

If you want to write your own plugin for cordlr, check out our guide.

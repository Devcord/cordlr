About

Cordlr is a modular approach for Discord Bots so Server owners can put together their own bots without coding any single line. The only requirement is a hosting service which is capable to host NodeJS apps with version 6.0 and higher.

Installation

Requirements

  • Node.js 6 >
  • Discord

Installation

In this demonstration we're using nave for managing the node environment.

  • nave use 6 - installs latest node version for major 6
  • npm install cordlr -g - installs the cordlr binary globally or npm install -g devcord/cordlr for the latest stable commit
  • mkdir mybot - create a folder for your bot
  • cordlr - inits the cordlr bot without any modules
  • cordlr install modulename - installs a cordlr module. is equal with npm install cordlr-modulename
  • Read the getting started on github
